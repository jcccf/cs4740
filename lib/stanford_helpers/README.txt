1.
Download the Stanford Parser, and the Stanford Named Entity Recognizer, and untar them somewhere.
http://nlp.stanford.edu/software/stanford-parser-2012-03-09.tgz
http://nlp.stanford.edu/software/stanford-ner-2012-04-07.tgz

2. 
Place parsetree.sh in the Stanford Parser folder
Place nes.sh in the Stanford NER folder

3. Adjust the location of these folders in Parser.py

4. Enjoy!
